# Movie App

This project is for practicing and understanding **Vite** and using the **Shadcn UI library**. The goal is to create a website that fetches and displays movie data from **The Movie Database (TMDb) API**.

## Features:

- Fetch movie data from TMDb API.
- Display the movie information using Shadcn UI components.
- Built with Vite for fast development and build processes.

## Technologies Used:

- **Vite** for fast bundling and development.
- **Shadcn UI** for UI components.
- **TMDb API** to fetch movie data.
